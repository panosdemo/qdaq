\doxysubsection{Detailed Description}
Class \mbox{\hyperlink{class_q_daq_object_model}{Q\+Daq\+Object\+Model}} represents the \mbox{\hyperlink{namespace_q_daq}{Q\+Daq}} object tree as a Qt item model.

This class enables the visual representation of the \mbox{\hyperlink{namespace_q_daq}{Q\+Daq}} object tree in Qt\textquotesingle{}s model/view classes.

The class is just a wrapper around the \mbox{\hyperlink{class_q_daq_object}{Q\+Daq\+Object}} hierarchy. No extra data is generated for the class.
