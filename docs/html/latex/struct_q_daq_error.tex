\doxysubsection{Detailed Description}
Structure containing error data.

It is emitted every time an error occurs in a \mbox{\hyperlink{class_q_daq_object}{Q\+Daq\+Object}}. Errors are gathered centrally on a \mbox{\hyperlink{class_q_daq_error_queue}{Q\+Daq\+Error\+Queue}} object that lives in \mbox{\hyperlink{class_q_daq_root}{Q\+Daq\+Root}}. The error queue is accesible through \mbox{\hyperlink{class_q_daq_object_a9f3d71496598a421c34fc099bcfcb1ae}{Q\+Daq\+Object\+::root()}}-\/$>$error\+Queue().
