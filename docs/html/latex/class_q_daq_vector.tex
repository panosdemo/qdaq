\doxysubsection{Detailed Description}
A buffer for storing double numbers. 

It is used for storing data from \mbox{\hyperlink{class_q_daq_channel}{Q\+Daq\+Channel}} objects.

The buffer has 3 modes, according to its Storage\+Type property\+: it can be circular, i.\+e., new data overwrite old data, it can have fixed size or it can be expandable.

Data are inserted at the end of the buffer by the function \mbox{\hyperlink{class_q_daq_vector_a35d662a68480a6cce746d50d48de5faa}{push()}} or the \mbox{\hyperlink{class_q_daq_vector_a41f1c54a437b60d16eff04ecc1cce4cb}{operator$<$$<$()}}. The contents can be read by the function \mbox{\hyperlink{class_q_daq_vector_a9784ab3060a24a6ecf0792d3153c3ecf}{get()}} or the \mbox{\hyperlink{class_q_daq_vector_ad203e0bf4a6f60ca09f0e3be7059511a}{operator\mbox{[}$\,$\mbox{]}()}}. The class provides read-\/only access to the data. It is not possible to change the value of a stored element.

The class defines functions for getting the min/max value, the mean and std deviation. The calculation of these quantities is stored internally and multiple calls to these functions do not cause a recalculation (except if the buffer contents have changed).

The buffer is explicitly shared, i.\+e., multiple instances share the same underlying data. This is used primarily for displaying real-\/time plots of data without copying the buffer.
