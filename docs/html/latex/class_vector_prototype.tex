\doxysubsection{Detailed Description}
The prototype for the Vector class. 

The Vector javascript class represents an array of floating point numbers. It is essentially a Qt\+Script wrapper for the \mbox{\hyperlink{class_q_daq_vector}{Q\+Daq\+Vector}} C++ class. Thus, when Qt-\/slots of \mbox{\hyperlink{namespace_q_daq}{Q\+Daq}} classes are called from Qt\+Script code, a \mbox{\hyperlink{class_q_daq_vector}{Q\+Daq\+Vector}} passed as an argument or as return value is converted to a JS object of class Vector.

Elements of a Vector can be accesed by the \mbox{[}\mbox{]} operator and the \char`\"{}length\char`\"{} property gives the number of elements.

New elements are added at the end of the Vector with the function \mbox{\hyperlink{class_vector_prototype_ab5444e1e14437663d0c8ad643f1c3483}{push()}}.

The property called \char`\"{}capacity\char`\"{} defines for how many elements there is memory allocated.

The \char`\"{}circular\char`\"{} property is a boolean value. If it is true then the Vector is a ring or circular buffer with maximum size equal to its capacity. In a circular vector, when the capacity has been reached, insertion of a new element causes deletion of the oldest element.

A Vector can be created in \mbox{\hyperlink{namespace_q_daq}{Q\+Daq}} scripts by the new operator in 3 possible ways\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{var x1 = new Vector(10); // An empty Vector of capacity 10}
\DoxyCodeLine{var x2 = new Vector([1,2,3]); // A Vector from a numeric Array}
\DoxyCodeLine{var x3 = new Vector(x2); // x3 is a copy of x2}
\DoxyCodeLine{x2[0] // returns 1}
\DoxyCodeLine{x3.length // returns 3}
\end{DoxyCode}


The \mbox{\hyperlink{class_vector_prototype}{Vector\+Prototype}} class defines a number of usefull functions for handling Vector objects in script code.
