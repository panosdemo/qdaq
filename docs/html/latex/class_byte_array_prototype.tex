\doxysubsection{Detailed Description}
The prototype for the Byte\+Array class. 

The Byte\+Array javascript class represents an array of bytes. It is essentially a Qt\+Script wrapper for the Q\+Byte\+Array C++ class. Thus, when Qt-\/slots of \mbox{\hyperlink{namespace_q_daq}{Q\+Daq}} classes are called from Qt\+Script code, a Q\+Byte\+Array passed as an argument or as return value is converted to a JS object of Byte\+Array class.

Elements of a Byte\+Array can be accesed by the \mbox{[}\mbox{]} operator and the \char`\"{}length\char`\"{} property gives the number of elements.

A Byte\+Array can be created in \mbox{\hyperlink{namespace_q_daq}{Q\+Daq}} scripts by the new operator in 3 possible ways\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{var b1 = new ByteArray(10); // An empty ByteArray of length 10}
\DoxyCodeLine{var b2 = new ByteArray("abcdef"); // A ByteArray from a string}
\DoxyCodeLine{var b3 = new ByteArray(b2); // b3 is a copy of b2}
\DoxyCodeLine{b2[3] // returns ASCII code of character 'c'}
\DoxyCodeLine{b3.length // returns 6}
\end{DoxyCode}


The \mbox{\hyperlink{class_byte_array_prototype}{Byte\+Array\+Prototype}} class defines a number of usefull functions for handling Byte\+Array objects in script code.
