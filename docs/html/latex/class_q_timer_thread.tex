\doxysubsection{Detailed Description}
A timer thread class. 

\mbox{\hyperlink{class_q_timer_thread}{Q\+Timer\+Thread}} inheritis Q\+Thread and uses internally Q\+Timer to implement its functionality.

After calling start() a thread is spawned and the function \mbox{\hyperlink{class_q_timer_thread_acf49e4e7ae236173a2d635ffed02cf72}{timer\+\_\+func()}} is called periodically with period equal to the interval property (in ms).

To use the class one has to subclass \mbox{\hyperlink{class_q_timer_thread}{Q\+Timer\+Thread}} reimplement \mbox{\hyperlink{class_q_timer_thread_acf49e4e7ae236173a2d635ffed02cf72}{timer\+\_\+func()}}. Note that \mbox{\hyperlink{class_q_timer_thread_acf49e4e7ae236173a2d635ffed02cf72}{timer\+\_\+func()}} is executed from the sepatrate timer thread thus and the user is responsible for synchronization issues.

The accuracy of the timer depends on Qt\textquotesingle{}s Q\+Timer class. According to their docs Q\+Timer uses the best OS timing routines in each case.
