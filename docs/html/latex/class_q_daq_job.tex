\doxysubsection{Detailed Description}
Base class for objects that perform a specific task reqursively.

Jobs can be organized as children of a parent-\/job or children of the super-\/class \mbox{\hyperlink{class_q_daq_loop}{Q\+Daq\+Loop}}.

When a \mbox{\hyperlink{class_q_daq_loop}{Q\+Daq\+Loop}} runs all its child jobs are executed by calling their \mbox{\hyperlink{class_q_daq_job_a39c072085e85df73a1608e9d347fb518}{exec()}} function. The \mbox{\hyperlink{class_q_daq_job_a39c072085e85df73a1608e9d347fb518}{exec()}} function first calls \mbox{\hyperlink{class_q_daq_job_afe30e7c3f03deb124ae4338984544a7f}{run()}} to perform the actual processing of this job (if there is any) and then calls \mbox{\hyperlink{class_q_daq_job_a39c072085e85df73a1608e9d347fb518}{exec()}} of each child job.

To give an example of the order that jobs are executed, the following job tree\+: \begin{DoxyVerb}job0
 |--job1
 |   |--job11
 |   |--job12
 |--job2
 |--job3\end{DoxyVerb}
 will be executed with the following order\+: job0-\/job1-\/job11-\/job12-\/job2-\/job3

Before the job can perform its task it must be \char`\"{}armed\char`\"{}. Arming does all the necessary initialization and is implemented in the function \mbox{\hyperlink{class_q_daq_job_a3af371946f6916812fd6ad03904b2293}{set\+Armed()}}.

{\footnotesize\ttfamily bool Q\+Daq\+Job\+::set\+Armed (\begin{DoxyParamCaption}\item[{bool}]{on }\end{DoxyParamCaption})\hspace{0.3cm}{\ttfamily [protected]}}



Arms or disarms a job. 

If on is true then \mbox{\hyperlink{class_q_daq_job_a3af371946f6916812fd6ad03904b2293}{set\+Armed()}} is called for each child-\/\+Q\+Daq\+Job (not \mbox{\hyperlink{class_q_daq_loop}{Q\+Daq\+Loop}}) and finally \mbox{\hyperlink{class_q_daq_job_aad493e9880d26c4b25cebe4bacee1717}{arm\+\_\+()}} is called to initialize this job. If any of those calls return false the arming is aborted and false is returned.

If on is false then \mbox{\hyperlink{class_q_daq_job_a5e6f51260c19f421bb4396720c2dc37b}{disarm\+\_\+()}} is called first and then \mbox{\hyperlink{class_q_daq_job_a3af371946f6916812fd6ad03904b2293}{set\+Armed()}} with parameter false is called for each child job.

This funtion is protected so that it cannot be called directly. \mbox{\hyperlink{class_q_daq_job}{Q\+Daq\+Job}} objects are armed by their parent \mbox{\hyperlink{class_q_daq_loop}{Q\+Daq\+Loop}}, when calling \mbox{\hyperlink{class_q_daq_loop_a87548dcf7b27cb4affbabd5908e0d17f}{Q\+Daq\+Loop\+::arm()}}.
