\doxysubsection{Detailed Description}
A class that provides storage of channel or other data. 

If \mbox{\hyperlink{class_q_daq_data_buffer}{Q\+Daq\+Data\+Buffer}} is used within a \mbox{\hyperlink{class_q_daq_loop}{Q\+Daq\+Loop}} then at each repetition it will store the values of all channels that have been assigned to it. The data are stored internally as a matrix of Q\+Daq\+Buffer objects, keeping one Q\+Daq\+Buffer per channel. A new row of data is added at each loop repetition.

\mbox{\hyperlink{class_q_daq_data_buffer}{Q\+Daq\+Data\+Buffer}} has an internal back buffer, where data generated in the loop thread are initially stored. The data is later transferred to the main buffer whenever possible and becomes available to the main application thread. The process is marshalled internally by a semaphore and the Qt signal/slot mechanism. Increasing the size of the back buffer can prevent data loss in fast loops.

The \mbox{\hyperlink{class_q_daq_data_buffer}{Q\+Daq\+Data\+Buffer}} may be also used as a static object outside of a loop. Data may be appended by the \mbox{\hyperlink{class_q_daq_data_buffer_ad612cb067f436d2003705d2220fd63fa}{push()}} function.
