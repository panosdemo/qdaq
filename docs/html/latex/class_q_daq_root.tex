\doxysubsection{Detailed Description}
\mbox{\hyperlink{namespace_q_daq}{Q\+Daq}} root object class.

At the top of the \mbox{\hyperlink{namespace_q_daq}{Q\+Daq}} tree structure there is an object of type \mbox{\hyperlink{class_q_daq_root}{Q\+Daq\+Root}}. It is a subclass of \mbox{\hyperlink{class_q_daq_object}{Q\+Daq\+Object}}.

The \mbox{\hyperlink{namespace_q_daq}{Q\+Daq}} root is a single static object which can be accessed by \mbox{\hyperlink{class_q_daq_object_a9f3d71496598a421c34fc099bcfcb1ae}{Q\+Daq\+Object\+::root()}}.

The \mbox{\hyperlink{namespace_q_daq}{Q\+Daq}} root should be instantiated in the main function, after Q\+Application.

The name of \mbox{\hyperlink{namespace_q_daq}{Q\+Daq}} root is \char`\"{}qdaq\char`\"{}.

A number of static items available to all \mbox{\hyperlink{namespace_q_daq}{Q\+Daq}} objects are defined under \mbox{\hyperlink{class_q_daq_root}{Q\+Daq\+Root}}\+:
\begin{DoxyItemize}
\item The error queue, available by \mbox{\hyperlink{class_q_daq_root_aabc3e6d597d770c3037b1da9c8f074d9}{error\+Queue()}}
\item a list of all application windows can be obtained by calling \mbox{\hyperlink{class_q_daq_root_a8327ee29571ce28f9a069d92c0e0f7b8}{daq\+Windows()}}
\item the I\+DE window can be obtained by \mbox{\hyperlink{class_q_daq_root_a15d85cbd31e3f7b566923b6cfd91f73e}{ide\+Window()}}
\item the root script session, \mbox{\hyperlink{class_q_daq_root_ae5bed36b118927b2c910e8d7dba3fc10}{root\+Session()}} 
\end{DoxyItemize}
