\doxysubsection{Detailed Description}
A class that represent a stream of numerical data, a signal. 

Real time data is handeled in Q\+Saq through the \mbox{\hyperlink{class_q_daq_channel}{Q\+Daq\+Channel}} class. E.\+g., an instrument provides measurement data through \mbox{\hyperlink{class_q_daq_channel}{Q\+Daq\+Channel}} objects.

\mbox{\hyperlink{class_q_daq_channel}{Q\+Daq\+Channel}} provides tools for displaying, transforming, averaging the data of the channel.
